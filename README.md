## Overview
Used a library  [ng2-pdf-viewer](https://github.com/VadimDez/ng2-pdf-viewer) to show pdf 

## Installation

```sh
$ git clone https://github.com/kbatyrbayev/ng-pdf-highlighter.git
$ cd ng-pdf-highlighter
$ npm install
$ ng serve --open
```
